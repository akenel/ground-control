Angel's own takeaway after building tig-tempest in one session: **"Start fresh with a Ground Control repo and you can build real stuff fast and never lose control."**

**Why:** A lost chat message kicked off the session; by the end there was a complete, faithful Tempest tribute (7 build steps, ~620 lines, 5 enemies, waves + level dive, superzapper, title screen, high scores, synthesized sound). Angel steered ("go, step N"), Tig rowed and verified each step, and every decision/step went to files ([[project-tig-tempest]], WORKLIST.md, CLAUDE.md). Human never lost the wheel — even in "the age of vibe coders with zero programming skills," Angel's engineering judgment did the steering.

**How to apply:** On any new build, set up the spine first (CLAUDE.md + WORKLIST.md + memory/), then work one small human-green'd step at a time. Trust the loop. See [[tempest-ruleset]] for how that discipline mapped onto a real feature roadmap.

*(Aside Angel shared: TEMPEST = both the literary storm mirroring inner turmoil AND the cybersecurity acronym — shielding devices from leaking data via stray emanations. A fitting name.)*
